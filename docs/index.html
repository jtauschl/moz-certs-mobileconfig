Mozilla Root Certificates

Keeps your device up to date with Mozilla's trusted root CA certificates — useful for systems that no longer receive Apple's own certificate updates.

↓ Download Profile

Installation

macOS
  1. Open the downloaded .mobileconfig file
  2. Go to System Settings → Privacy & Security → Profiles
  3. Double-click the pending profile and confirm installation
iOS / iPadOS
  1. Tap the download link above in Safari
  2. Go to Settings → General → VPN & Device Management
  3. Tap the profile and confirm installation
